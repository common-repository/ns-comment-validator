Client side validation of your WordPress Comment form

== Description ==

This plugin adds client side validation in WordPress Comment form. This acts as a level of security also. Without entering valid input user wont be able to post comment. Works for both loggedin and non-loggedin users.
This plugin uses [jQuery Validation Plugin](http://jqueryvalidation.org/)


== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'ns-comment-validator.zip' from your computer
1. Upload
1. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =

1. Extract 'ns-comment-validator.zip' to your computer
1. Upload the 'ns-comment-validator' directory to your 'wp-content/plugins' directory
1. Activate the plugin on the WordPress Plugins dashboard
